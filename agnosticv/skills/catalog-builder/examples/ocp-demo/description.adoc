Generative AI Demo with Red Hat OpenShift AI – Experience the power of Generative AI on Red Hat OpenShift AI using Model as a Service (MaaS) deployed on optimized virtualized infrastructure, without requiring any local GPU resources.

This immersive demo environment showcases how OpenShift AI seamlessly integrates with OpenShift components like Pipelines, GitOps, and Data Foundation to enable enterprise-scale AI workflows on hybrid cloud infrastructure. It highlights secure, scalable, and efficient AI deployment using virtualized compute resources powered by MaaS for inference.

== Key Components
* *Red Hat OpenShift Container Platform* – Secure and stable foundation for containerized workloads.
* *Red Hat OpenShift AI* – Comprehensive platform for building, training, and serving AI/ML models at scale.
* *Red Hat OpenShift Data Foundation* – Resilient, scalable storage for AI workloads.
* *Red Hat OpenShift Pipelines* and *GitOps* – Automate and streamline AI lifecycle management.
* *Red Hat OpenShift Serverless* and *Service Mesh* – Simplify scalable, secure AI microservices deployment.

== Special Features
* *Model as a Service (MaaS)* – Access and serve AI models without deploying GPUs locally.
* *Hybrid Cloud Integration* – Demonstrates Red Hat’s unified AI platform across data science and infrastructure teams.
* *Generative AI in Action* – Explore real-world enterprise use cases powered by OpenShift AI and integrated MLOps pipelines.

== Provisioning and Access
* *Access Details*: Credentials for OpenShift AI and the cluster will be provided once the demo environment is ready.
