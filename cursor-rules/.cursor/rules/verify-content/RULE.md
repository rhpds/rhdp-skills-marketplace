# Content Verification Skill

When user says ANY of these phrases, invoke this skill:
- "verify content"
- "validate content"
- "check content quality"
- "review content"
- "validate workshop"
- "verify workshop"
- "check lab quality"

## Skill Execution

**Action**: Read and follow `~/.cursor/skills/verify-content/SKILL.md` completely.

**OR if skills are in Claude directory**: Read and follow `~/.claude/skills/verify-content/SKILL.md` completely.

- Know/Do/Check structure compliance
- Red Hat branding standards
- Navigation and metadata completeness
- Content quality and clarity
