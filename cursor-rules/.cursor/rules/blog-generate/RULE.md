# Blog Post Generator Skill

## Trigger Commands

When user says ANY of these phrases, invoke this skill:
- "generate blog"
- "create blog post"
- "convert to blog"
- "transform to blog"
- "blog from workshop"
- "blog from lab"
- "create blog from content"

## Skill Execution

**Action**: Read and follow `~/.cursor/skills/blog-generate/SKILL.md` completely.

**OR if skills are in Claude directory**: Read and follow `~/.claude/skills/blog-generate/SKILL.md` completely.

## What This Skill Does

Transforms Red Hat Showroom workshop content into blog posts:
- Converts AsciiDoc format
- Adapts Know/Do structure for blog narrative
- Adds Red Hat Developer blog metadata
- Optimizes for blog readability
- Maintains technical accuracy
