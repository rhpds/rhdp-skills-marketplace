# Demo Content Generator Skill

## Trigger Commands

When user says ANY of these phrases, invoke this skill:
- "create demo"
- "generate demo"
- "new demo"
- "create presentation"
- "generate presenter-led demo"
- "create showroom demo"

## Skill Execution

**Action**: Read and follow `~/.cursor/skills/create-demo/SKILL.md` completely.

**OR if skills are in Claude directory**: Read and follow `~/.claude/skills/create-demo/SKILL.md` completely.

## What This Skill Does

Creates Red Hat Showroom presenter-led demos with:
- AsciiDoc formatted content
- Know/Show structure
- Demo module organization
- Presenter notes and talking points
- Red Hat branding standards
