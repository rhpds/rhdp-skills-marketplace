# Updating Screenshots

This directory contains screenshots for the updating guide at `/docs/setup/updating.html`.

## Required Screenshots

Please add the following screenshots from the Claude Code `/plugin` interface:

1. **step1-plugin-marketplaces.png**
   - Shows the `/plugin` interface with Marketplaces tab selected
   - Displays rhdp-marketplace with options: "Update marketplace", "Enable auto-update", "Remove marketplace"

2. **step2-updated-marketplace.png**
   - Shows the Marketplaces tab after running "Update marketplace"
   - Displays "âœ“ Updated 1 marketplace" confirmation message

3. **step3-installed-tab.png**
   - Shows the Installed tab with showroom plugin details
   - Displays "Version: 2.4.3" (or current version before update)
   - Shows full plugin metadata: Scope, Author, Status, Installed components

4. **step4-update-now.png**
   - Shows plugin detail view with "Update now" option highlighted
   - Menu options visible: "Disable plugin", "Mark for update", "Update now", "Uninstall"

5. **step5-version-updated.png**
   - Shows the Installed tab after update
   - Displays updated version number (e.g., "Version: 2.4.4")

## Screenshot Requirements

- **Format**: PNG
- **Dimensions**: Full width of terminal/UI
- **Background**: Dark terminal theme (as shown in current screenshots)
- **Quality**: High resolution, clear text

## How to Capture

1. Run `/plugin` in Claude Code
2. Use macOS Screenshot (Cmd+Shift+4) or similar tool
3. Save with exact filenames listed above
4. Place in this directory
