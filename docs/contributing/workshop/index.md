# Workshop: Use Claude to Create Custom Skills & Plugins

Learn to **use Claude effectively** to create custom skills and plugins by leveraging real RHDP examples as templates.

---

## Workshop Modules

**Module 1: [What Are Skills & How They Work](module-1.html)**
- Understanding skills and plugins
- How Claude Code loads and uses skills
- RHDP marketplace structure
- **Time:** 10 minutes

**Module 2: [Use Claude to Analyze Existing Skills](module-2.html)**
- Ask Claude to explain showroom and agnosticv skills
- Learn patterns from real examples
- Understand what makes a good skill
- **Time:** 15 minutes

**Module 3: [Use Claude to Create Your First Skill](module-3.html)**
- Effective prompting techniques
- Convert your workflow into a skill with Claude
- Iterate and improve with Claude's help
- **Time:** 30 minutes

**Module 4: [Use Claude to Package as Plugin](module-4.html)**
- Generate plugin.json with Claude
- Create proper directory structure
- Add documentation
- **Time:** 15 minutes

**Module 5: [Testing & Publishing](module-5.html)**
- Test your Claude-generated skill
- Fix issues by asking Claude
- Publish to marketplace
- **Time:** 20 minutes

---

## Prerequisites

- Claude Code CLI or VS Code with Claude extension installed
- A workflow or task you want to automate
- Git installed
- That's it! Claude will write the code.

---

## Learning Approach

This workshop teaches you to **leverage Claude as your skill developer**:

1. **Claude reads examples** - Point Claude to RHDP skills
2. **Claude learns patterns** - Ask Claude to analyze and explain
3. **Claude writes your skill** - Give Claude your requirements
4. **Claude iterates** - Ask Claude to improve and fix
5. **You test and guide** - Validate Claude's work and request changes

**You provide the requirements, Claude does the coding.**

---

## What You'll Learn

By the end of this workshop, you'll know how to:

- ✅ Prompt Claude to analyze existing skills
- ✅ Describe your workflow for Claude to convert
- ✅ Iterate with Claude to refine the skill
- ✅ Have Claude package everything properly
- ✅ Create unlimited skills using these techniques

**Example:**
```
You: "I have a shell script that deploys apps to OpenShift.
Can you read agnosticv/skills/catalog-builder/SKILL.md and create
a similar skill for my deployment workflow?"

Claude: [Analyzes example, creates your skill]

You: "Add validation for prerequisites and better error messages"

Claude: [Updates skill]
```
